Artist Statement

The work uses in the end is a Raspberry Pi Pico W, reset button, Non-Contact Water/Liquid Level Sensor and DFPlayer Mini MP3 Player with a micro SD card. The project is soldered on a perma board which was educational. The water sensor was the focus of the work. It uses the “inductive capacitance of water” to see if water is at the level of the sensor on the container. A positive is that you can use most containers that are not metal for the sensor. So, the user can use different cups as long as the sensitivity of the sensor is adjusted. It lets the user measure water without sight needed. The display was added to the work but did not work so is not included in the performance. The hardest part is the perma board and display. The perma board does not let you easily adjust a wrongly put wire is one downside of that. On the other hand, it feels more stable after it works. The DFplayer Mini Mp3 player was great as long as the wires were placed correctly, which was messed up for a bit when transferred to the Perma board. The performance met the goal I had of the sensor telling the user to stop at the right point with noise.
